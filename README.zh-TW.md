## 🎹 鍵盤操作說明

> 本工具為終端機介面，全程鍵盤操作，不需要滑鼠。

| 操作 | 按鍵 | 說明 |
|------|------|------|
| 移動選項 | `↑` / `↓` | 上下切換選項 |
| 勾選 / 取消勾選 | `Space` 空白鍵 | 多選模式（checkbox）下使用 |
| 確認 | `Enter` | 確認選擇或送出輸入 |
| 接受預設值 | `Enter`（不輸入）| 直接按 Enter 使用括號內預設值 |
| 確認執行 | `Y` + `Enter` 或直接 `Enter` | 預設為 Yes |
| 取消 / 放棄 | `N` + `Enter` | |

## 🚀 快速啟動

### Mac
雙擊 `start.command`

> 第一次執行會自動安裝所需套件，稍等片刻即可。

### Windows（第一次安裝）

1. 前往 GitHub 下載專案 ZIP：
   **Code → Download ZIP** → 解壓縮到任意資料夾
2. 雙擊 `start.bat`
   - 若尚未安裝 Python，會**自動安裝**（需要網路）
   - 安裝完成後，依提示**再雙擊一次** `start.bat`
   - 自動建立環境、安裝套件，完成後進入程式

> 之後每次只要雙擊 `start.bat` 即可。
>
> 建議使用 **Windows Terminal**（Win 11 預設）而非傳統 CMD，顯示效果較佳。

啟動後，用 `↑` `↓` 選擇功能，按 `Enter` 確認：

```
請選擇功能：
 » 新建銷售單
   建立出貨單（銷貨單拋轉）
   建立出庫單（出貨單拋轉）
   匯出庫存報表（Excel）
   退出
```

> 任何步驟都可以選「← 返回」或按 `N` 退回上一步，執行前也會再次確認。

## 🛠️ 功能一覽

| 功能 | 說明 |
|------|------|
| 新建銷售單 | 把客戶寄來的採購單 Excel，自動建成 Ragic 銷貨單 |

## 🤖 Agent mode

以自然語言操作 ERP 資料，由 Claude AI 驅動。

**▸ 即時查詢 (Real-time Query)**
```
「TW01 現在還有多少 BBB042？」
「幫我查所有倉庫的庫存狀況」
```

**▸ 數據決策 (Intelligence & Analysis)**
```
「1～3 月銷售前五名 SKU 是哪些？」
「分析 XXX 客戶的訂單趨勢與缺貨風險」
```

**▸ 供應鏈優化 (Supply Chain Suggestion)**
```
「BBB042 建議補貨量（基於過去 3 個月銷量）」
「列出當前庫存不足的清單」
```

**▸ 自動化報表 (Report Automation)**
```
「整理本月未出貨訂單並存至 Excel」
「匯出庫存低於預警線的商品清單」
```

**設定方式：** 需要 [Anthropic API Key](https://console.anthropic.com)，第一次進入 Agent mode 時會提示輸入，自動儲存於本機，之後不需再輸入。

> 在 Agent mode 內輸入 `重設 key` 可更換 API Key。

## 📖 功能詳細說明

### 1. 新建銷售單

**前置作業：** 將客戶採購單 Excel 放入對應資料夾

```
client_order/
  ├── LE/         ← 格式 A 採購單
  ├── TRU/        ← 格式 B 採購單
  └── TEMPLATE/   ← 通用模板採購單
```

**Step 1｜選擇採購單**（多選）
```
請選擇要處理的採購單（Space 勾選，Enter 確認）：
 ◯ LE/client_order_0324.xlsx
 ◯ TRU/TRU-20260402.xlsx
```
- `Space` 勾選，可一次選多個
- 不選直接 `Enter` → 返回主選單

**Step 2｜防重複檢查**（自動）
```
⚠ 此訂單已於 2026/04/05 上傳（Ragic ID: 12345）
是否跳過（建議跳過以避免重複）？ [Y/n]
```

**Step 3｜客戶比對**（自動）
- 系統根據門市代碼自動比對客戶
- 找到多筆時，`↑` `↓` 選擇，`Enter` 確認
- 找不到時，可輸入關鍵字搜尋，或使用暫代客戶

**Step 4～9｜訂單設定**

| 步驟 | 選項 |
|------|------|
| 訂單單別 | 一般 / 公關品 / 樣品 / 蝦皮 / 官網 |
| 訂單狀態 | 未出貨 / 預接單 / 已收款未出貨 / 已出貨未收款 |
| 稅率 | 5%（外加）/ 5%（內含）|
| 運費 | 輸入數字，直接 `Enter` = 預設 0 |
| 業務分潤 | 8% / 2% / 無 |
| 備註 | 輸入文字，直接 `Enter` 略過 |

**Step 10｜預覽與確認**
```
商品明細：
  薇薇內心世界系列毛絨盲盒  × 10  @  199.5  =  1,995
  ...

訂單摘要：
  客戶：XXX 門市    稅率：5% 外加
  小計：1,995       稅額：100       運費：0       總計：2,095

確認送出此訂單？ [Y/n]
```

**結果：** Ragic 自動建立銷貨單，採購單檔案移至 `done/` 資料夾

---

### 2. 建立出貨單（銷貨單拋轉）

**Step 1｜選擇銷貨單**（多選）
```
請選擇要拋轉的銷貨單（Space 勾選，Enter 確認）：
 ◯ SO-20260405-001  XXX 門市  2026/04/05  [未出貨]
 ◯ SO-20260405-002  OOO 門市  2026/04/04  [預接單]
```

**Step 2｜確認執行**
```
── 即將執行：建立出貨單 ──
  SO-20260405-001  XXX 門市
  SO-20260405-002  OOO 門市

確認執行？ [Y/n]
```

**結果：** Ragic 自動生成出貨單（DO 編號）

---

### 3. 建立出庫單（出貨單拋轉）

**Step 1｜選擇出貨單**（多選）
```
請選擇要拋轉的出貨單（Space 勾選，Enter 確認）：
 ◯ DO-20260405-001  XXX 門市  2026/04/05
```

**Step 2｜選擇倉庫**（單選）
```
請選擇倉庫：
 » TW01  台灣倉庫
   TW02  新竹倉庫
   ← 返回
```
- 選「← 返回」→ 回到 Step 1 重新選出貨單

**Step 3｜庫存編號比對**（自動 + 必要時手動）
```
只有一筆，自動帶入：
  BBB042 薇薇系列 → INV-001（自動）

有多筆，需手動選擇：
  請選擇 BBB042 薇薇系列 的庫存編號：
   » INV-001
     INV-002
     ← 返回（回到 Step 2 重新選倉庫）

無庫存，自動略過：
  ⚠ BBB042 在 TW01 無庫存紀錄，略過
```

**Step 4｜確認執行**
```
── 即將執行：建立出庫單 ──
  DO-20260405-001  XXX 門市
  倉庫：TW01  台灣倉庫
  BBB042 → INV-001

確認執行？ [Y/n]
```

**結果：** Ragic 自動生成出庫單，並自動補填倉庫代碼與庫存編號

---

### 4. 匯出庫存報表（Excel）

**Step 1｜選擇倉庫**（單選）
```
請選擇倉庫：
 » TW01  台灣倉庫
   TW02  新竹倉庫
   ← 返回
```

**Step 2｜選擇模板**（多選）
```
請選擇模板（Space 勾選，Enter 確認）：
 ◯ quote-template.xlsx
 ◯ inventory-template.xlsx
```

**Step 3｜確認執行**
```
── 即將執行：匯出庫存報表 ──
  倉庫：TW01  台灣倉庫
  模板：quote-template.xlsx

確認執行？ [Y/n]
```

**結果：** 輸出至 `exports/` 資料夾，檔名格式：`{模板名}_{倉庫}_{日期}_{時間}.xlsx`

## 📋 模板說明

| 模板 | 用途 | 現貨自動填入 | 參考圖 |
|------|------|:-----------:|:------:|
| `quote-template.xlsx` | 報價單，客戶於 S001~S005 欄填入各門市訂購數量 | ✓ | ✓ |
| `inventory-template.xlsx` | 庫存總覽，顯示現貨與在途數量 | ✓ | ✓ |

**quote-template 欄位說明：**

| 欄位 | 說明 |
|------|------|
| D 欄 | 國際條碼（程式比對用，請勿修改） |
| N 欄 | 台灣現貨（匯出時自動填入） |
| O 欄 | 訂單總數（自動加總 S001~S005） |
| P~T 欄（S001~S005）| 客戶自填門市店號與各門市訂購數量 |

## 📁 檔案架構

```
erp-tools-ragic/
├── start.command         ← Mac 啟動腳本（雙擊執行）
├── start.bat             ← Windows 啟動腳本（雙擊執行）
├── client_order/         ← 放入客戶採購單
│   ├── LE/               ← 格式 A
│   ├── TRU/              ← 格式 B
│   └── TEMPLATE/         ← 通用模板
├── templates/            ← 庫存報表模板（請勿手動修改）
│   ├── quote-template.xlsx
│   └── inventory-template.xlsx
├── exports/              ← 匯出的庫存報表（程式自動產生）
├── logs/                 ← 操作日誌，每日一份
└── upload_log.json       ← 防重複上傳記錄
```
